## 진행 방법

- 리팩터링 2판을 보고 책에 나와 있는 예제를 `Kotlin` 으로 작성
- 각 챕터별로 `refactoring/chxx` 형식의 브랜치를 만들고, 해당 브랜치에 예제 작성
- 학습 기록은 각 예제 브랜치의 `chxx.md`에 작성한다.

> 아직 구체적인 스터디 진행 방법은 나오지 않아 추후 작성 예정
